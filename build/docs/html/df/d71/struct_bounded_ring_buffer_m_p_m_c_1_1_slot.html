Detailed Description

template<typename T>
struct BoundedRingBufferMPMC< T >::Slot

Individual slot in the ring buffer.

Each slot is aligned to cache line boundary to prevent false sharing between adjacent slots.

Each slot contains a flag indicating if data is present and the actual data storage. The slot is aligned to cache line boundary to prevent false sharing between adjacent slots.
